# Weather App

This is a simple Weather App developed using Python that fetches real-time weather information for any city using the OpenWeatherMap API.

The project allows users to enter a city name and view the current temperature and humidity. It helped me understand how APIs work and how Python can be used to retrieve and display live data from the internet.

## Features

- Accepts city name as input
- Fetches real-time weather data
- Displays current temperature
- Displays humidity level
- Handles invalid city names and API errors

## Technologies Used

- Python
- VS Code
- OpenWeatherMap API
- Git & GitHub

## Python Libraries Used

- requests

## How It Works

1. The user enters a city name.
2. The program sends a request to the OpenWeatherMap API.
3. The API returns weather data in JSON format.
4. The program extracts temperature and humidity information.
5. The weather details are displayed to the user.

## Sample Output

Enter city name: Hyderabad

Temperature: 34 °C
